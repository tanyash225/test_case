## Задание 2
```auto_dock.py``` - скрипт для конвертации более распространенных структурных файлов (SMI и SDF) в формат PDBQT

Установка зависимостей

```pip install -r requirements.txt```

```out_from_smi_pdbqt``` - примеры выходных файлов на нескольких молекулах в виде SMILES из примеров

пример использования скрипта из SMILES представления молекул с включением опции "гибких колец":

``` python auto_dock_2.py example.smi -o out_pdbqt --flexible-rings-6plus ```
